## **Smart Text Editor using Data Structures**

The **Text Editor using Data Structures** is a console-based software application designed to manage and manipulate text efficiently using fundamental data structures. It allows users to insert, delete, edit, replace, and display words while supporting undo and redo operations. By using linked lists, stacks, and queues, the system ensures structured text handling, prevents errors, and demonstrates real-world applications of data structures in an organized and efficient manner.

# Text Editor Using Data Structures

## Project Summary
This project is a console-based text editor implemented in C++ using core Data Structures.
It allows inserting, deleting, editing, undo/redo operations and word analysis.

## Data Structures Used
- Singly Linked List (text storage)
- Stack (Undo / Redo operations)
- Queue (History tracking and FIFO display)

## Key Features
- Insert words normally or at specific positions
- Delete words using linked list and queue
- Undo and redo editing operations
- Replace and edit words
- Display text using Queue (FIFO)
- Word statistics (count, longest, shortest, average length)

## How to Run
1. Compile:
   g++ text_editor.cpp -o editor
2. Run:
   ./editor
